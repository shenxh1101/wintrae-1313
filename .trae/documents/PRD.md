## 1. 产品概述

露营装备清单规划工具是一款面向露营爱好者的纯前端网页应用，帮助用户根据出行条件智能规划装备清单，避免遗漏、超重和重复携带问题。

- 核心价值：让露营装备准备更科学、更省心，提升露营体验
- 目标用户：周末露营爱好者、户外探险者、露营组织者
- 解决问题：装备遗漏、重量超标、分工不清、易耗品数量不足

## 2. 核心功能

### 2.1 功能模块

1. **目的地信息区**：季节、天数、天气、营地类型设置
2. **同行人员区**：人数管理、人员信息编辑
3. **装备库区**：分类装备浏览、勾选添加、共享/个人标记
4. **清单生成区**：自动生成清单、智能提示、重量统计
5. **重量概览区**：总重量、人均重量、分类重量统计

### 2.2 页面详情

| 模块名称 | 功能描述 |
|-----------|----------|
| 目的地信息 | 选择季节（春/夏/秋/冬）、设置天数（1-7天）、选择天气（晴/雨/雪）、选择营地类型（山野/湖边/营地/沙漠） |
| 同行人员 | 添加/删除人员、设置人员姓名、分配背包承重能力 |
| 装备库 | 按类别展示装备（帐篷、炊具、照明、急救、衣物、食品、其他）、支持搜索筛选、勾选加入清单、标记共享或个人携带 |
| 清单生成 | 显示已选装备列表、调整数量、分配携带者、智能提示（遗漏类别、重复携带、重量超限、易耗品数量） |
| 重量概览 | 总重量展示、人均重量、分类重量饼图/柱状图、重量进度条 |
| 背包分配 | 按背包分配装备、拖拽调整、显示每个背包的重量 |
| 方案管理 | 保存多个方案、切换方案、删除方案、导出方案 |
| 检查模式 | 出发前逐项检查、勾选确认、进度展示 |

## 3. 核心流程

用户进入页面 → 设置目的地信息（季节/天数/天气/营地类型）→ 添加同行人员 → 从装备库勾选所需装备并标记共享/个人 → 查看智能提示并调整清单 → 查看重量概览 → 按背包分配装备 → 保存方案 → 出发前检查模式逐项确认 → 导出清单

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：森林绿（#2D5A27）作为主色，配合土棕色（#8B7355）和米白色（#F5F2EB）
- **辅助色**：橙色（#E67E22）作为强调色，用于提示和重要操作
- **设计风格**：户外自然风格，卡片式布局，圆角设计，带有轻微的纸张纹理感
- **字体**：使用系统无衬线字体，标题加粗，正文清晰易读
- **图标风格**：线性图标，与自然/露营主题契合
- **整体氛围**：温暖、自然、专业，给人可靠的户外装备专家感觉

### 4.2 页面设计概览

| 模块名称 | UI 元素 |
|-----------|--------|
| 顶部标题区 | Logo + 标题 + 方案切换下拉 + 保存按钮 |
| 目的地信息卡片 | 四个选择器横排，图标+下拉选择，卡片式设计 |
| 同行人员卡片 | 人员列表，头像+姓名+承重，添加按钮 |
| 装备库卡片 | 分类标签页，装备网格列表，复选框+名称+重量+共享标记 |
| 清单生成卡片 | 装备列表，可折叠分类，数量调整，携带者分配 |
| 智能提示区 | 提示卡片，不同类型不同颜色（警告橙、信息蓝、成功绿） |
| 重量概览卡片 | 大号数字展示总重量，进度条，分类重量条形图 |
| 底部操作栏 | 保存方案、导出清单、检查模式切换按钮 |

### 4.3 响应式设计

- 桌面端（≥1024px）：左右两栏布局，左侧装备库，右侧清单+概览
- 平板端（768-1023px）：上下堆叠，标签页切换
- 移动端（<768px）：单列布局，折叠卡片，底部导航切换

### 4.4 交互动效

- 卡片悬停：轻微上浮+阴影加深
- 勾选动画：缩放+颜色过渡
- 提示出现：滑入+淡入
- 重量变化：数字滚动动画
- 检查模式：进度条平滑过渡
